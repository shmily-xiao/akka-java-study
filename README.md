### CPI :

指令平均周期数（英语：Cycle Per Instruction, CPI），也称每指令周期，即执行在计算机体系结构中一条指令所需要的平均时钟周期（机器主频的倒数）数.
一个完整的计算机系统执行时间，即指令周期指从内存中取出并执行该条指令所用的全部时间，它取决于硬件结构和各部件的性能.
